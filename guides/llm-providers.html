LLM Providers

Configure OpenAI, Ollama, Vertex AI, Bedrock, Anthropic, and other LLM providers in Recif.

5 min read

Overview#

Recif is LLM-agnostic. Corail provides a unified LLMProvider interface that abstracts away the differences between providers. You can switch providers by changing a few lines of YAML — no code changes required.

Supported providers:

ProviderStatusLocalAPI KeyModels
OpenAIStableNoRequiredGPT-4o, GPT-4o-mini, o1, o3
OllamaStableYesNoneLlama 3, Mistral, Gemma, Qwen
