## Overview

This page documents all environment variables that configure Recif platform components and Corail agent runtime. Variables are organized by component.

> [!NOTE] This page is under active development. Full reference coming soon.

## Categories

- **Recif API Server** — database connection, auth secrets, MLflow URL, gRPC ports
- **Recif Operator** — reconciliation interval, namespace watch, CRD settings
- **Corail Runtime** — LLM config, tool endpoints, memory backend, channel ports
- **Maree Ingestion** — embedding model, vector store, staging config
