## Overview

Recif exposes both REST and gRPC APIs. The REST API is used by the dashboard and CLI, while gRPC handles real-time communication between Recif and Corail agent pods.

> [!NOTE] This page is under active development. Full API reference coming soon.

## API Groups

- **Agents** — CRUD operations, status, release management
- **Evaluation** — trigger evals, get results, manage datasets
- **Governance** — risk profiles, guards, audit logs
- **Deployments** — canary releases, traffic splitting, rollback
- **Knowledge Bases** — CRUD, ingestion triggers, evaluation results
- **gRPC Control Plane** — bidirectional streaming between Recif and Corail pods
