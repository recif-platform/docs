## Overview

Recif leverages Istio traffic splitting to provide production-safe deployment strategies for your agents. Roll out new versions gradually, test with a subset of traffic, and automatically roll back if quality drops.

- Blue/green deployments for zero-downtime switches
- A/B testing between agent versions
- Auto-rollback based on evaluation scores
- Flagger webhook integration for automated promotion
- Monitoring deployment health in the dashboard
