## Overview

Creating an agent in Recif involves defining a declarative YAML configuration (Agent CRD), setting up its LLM provider, configuring channels, and applying governance rules. This guide walks through the entire process.

> [!NOTE] This page is under active development. Full content coming soon. For now, see the [Quickstart](/docs/quickstart) for a fast-track setup.

## What You'll Learn

- How to write an Agent CRD from scratch
- Configuring the Corail runtime strategy (ReAct, custom)
- Connecting channels (REST API, widget, Slack)
- Wiring tools and MCP servers
- Setting up SOUL.md and AGENTS.md for agent identity
- Applying governance rules and guards
