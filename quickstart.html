Quickstart

Install Recif and deploy your first AI agent in under 5 minutes.

3 min read

Prerequisites#

Before you begin, make sure you have:

  • Kubernetes cluster (v1.28+) — local (Colima, Minikube, kind) or cloud (GKE, EKS, AKS)
  • Helm (v3.12+)
  • kubectl configured for your cluster
  • An LLM provider API key (OpenAI, or a local Ollama instance)

Tip

For local development, we recommend Colima with Kubernetes enabled. It gives you a full K8s cluster on your Mac with minimal setup.

  • Istio configuration (if Istio is installed)

The evaluation pipeline will:

  1. Run your agent against the test dataset
  2. Score responses using MLflow GenAI scorers
  3. Compare against quality thresholds
